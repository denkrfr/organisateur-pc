## v1.1.3 — Petits groupes : cases à cocher directement visibles

### Ce qui change

Avant : pour 2 fichiers, fallait cliquer "Voir / Modifier" → fenêtre s'ouvre → cocher → valider. 5 clics pour rien.

Maintenant :
- **Groupes ≤ 4 fichiers** : tout affiché directement avec checkbox individuelle visible. **1 clic** pour cocher la copie à supprimer. C'est tout.
- **Groupes > 4 fichiers** : comportement actuel conservé (bouton "Voir / Modifier" pour le détail, car ça surchargerait l'écran sinon).

### Installation

1. Télécharge `Organisateur-v1.1.3-portable-windows.zip` (~292 Mo)
2. Clic-droit → Extraire tout
3. Double-clic sur `Organisateur.exe`
